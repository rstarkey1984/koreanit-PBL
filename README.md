# PBL 서버 프로그램 프로젝트

이 프로젝트는  
**PBL(Project Based Learning) 방식으로 서버 프로그램 개발 전체 흐름을 학습**하기 위한 교육용 프로젝트이다.

요구사항 분석부터 설계, 통합 구현, 테스트, 배포까지  
실제 서비스 개발 과정을 단계별로 경험하는 것을 목표로 한다.

---

## 프로젝트 특징

- 국가기술자격 및 NCS 실무 평가 흐름을 참고한 PBL 구성
- Spring Boot 기반 서버 프로그램
- SQL 중심 Repository 설계 (ORM 미사용)
- 메인 서버 + 배치/워커 프로세스 분리
- Docker 기반 실행 및 배포 환경

---

## 대상 학습자

- 서버 프로그램 입문~중급 학습자
- 실무 흐름을 경험하고 싶은 개발자
